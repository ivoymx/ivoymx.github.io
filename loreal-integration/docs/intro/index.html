    Welcome to this documentation. This integration is a bridge for our SameDay API using a SOAP protocol.

    Base URL

    The first step thats you need is know the base URL to connect which depends of the enviroment.

    Testing enviroment

    https://loreal.ws.ivoy.dev/

    Production enviroment

    https://loreal.ws.ivoy.mx/

    NOTE: The URL of production enviroment maybe change in the futher.
